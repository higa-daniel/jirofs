# Menu item images

Add image files here to show icons/photos on menu items.

**Nori Fries:** add a file named `nori_fries.png` (or `nori_fries.jpg`) and ensure the Nori Fries item in `lib/data/sample_menu.dart` has:

- `imageUrl: 'asset:assets/items/nori_fries.png'` (use `.jpg` if your file is JPG).

For other items, add the file and set `imageUrl: 'asset:assets/items/your_file.png'` on the corresponding `MenuItem`.
